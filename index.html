COIN FLIP SIMULATOR

This simulator runs how long it takes to get to a cerain number of tails in a coin flip. If you type in 100, it will return a the number of coin flips needed to get 100 tails from a coin flip.

The second simulator checks how long it takes to get a certain number of tail flips in a row. If you type 10, the simulator will tell you how long it took to get 10 tail flips in a row.
